Real-Time Vulkan Ray Tracer

This project is a custom-built real-time ray tracer using Vulkan and C++. It renders physically accurate lighting effects such as reflections, shadows, and global illumination using Vulkan's KHR_ray_tracing_pipeline and KHR_acceleration_structure extensions.

  • 🚀 Performance-Oriented: Acceleration structures (BLAS/TLAS) are built on-the-fly with GPU-side compaction and memory reuse.
  • 📦 Model Support: Parses OBJ files with materials and textures. Supports arbitrarily large triangle meshes, loaded into GPU-managed buffers.
  • 🎮 Interactive: Real-time camera navigation (orbit, pan, zoom) with support for animated instances and live hot reloading of scenes.
  • 📊 Shader Binding Table: Implements a programmable SBT layout to map rays to hit shaders and material indices with custom layout logic.
  • 🌐 Material System: GGX-based microfacet BRDF with energy-conserving reflection model, optional Fresnel, and roughness remapping.
  • 🌀 Temporal Denoising: Integrates a simplified reprojection-based denoiser to reduce noise over time without ghosting.

This ray tracer was built from scratch as a deep dive into GPU ray tracing and modern graphics APIs. It helped me develop a practical understanding of:

  • Memory management and synchronization in Vulkan
  • Pipeline creation and descriptor set layout for complex shaders
  • Spatial data structures like BVHs (Bounding Volume Hierarchies)
  • Deferred lighting techniques for ray-based rendering
  • Low-level debugging using RenderDoc and NVIDIA Nsight

👉 Try It: Clone and build instructions available on GitHub (repo is private by default, but sharable on request). Built with cross-platform support in mind (Linux & Windows).
